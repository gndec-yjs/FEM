Spring and Bar Elements

Spring and bar (axial) elements are the simplest finite elements. They illustrate the derivation of element stiffness matrices, nodal force–displacement relations, and the assembly of global stiffness for connected elements. The following text uses plain HTML and monospaced blocks so matrices and equations appear exactly as in book-style text.

1. Spring Element (2-node)

Consider a linear spring of stiffness k connecting node 1 and node 2. The nodal displacements are u1 and u2.

Hooke's law (axial spring):

    f = k * (u2 - u1)

Nodal force vector (assembled):

[ f1 ]   =  k * [  1  -1 ] [ u1 ]
[ f2 ]          [ -1   1 ] [ u2 ]

Therefore the element stiffness matrix is:

[k_e] = k * [  1  -1
              -1   1 ]

2. Axial Bar Element (1D)

A straight bar element resists only axial deformation. For a linear elastic bar with cross-sectional area A, Young's modulus E, and length L, the axial strain, stress and internal force are: 

Strain:   epsilon = (u2 - u1) / L

Stress:   sigma = E * epsilon = E * (u2 - u1) / L

Axial force in element:  F = A * sigma = (E * A / L) * (u2 - u1)

Nodal force–displacement relation (matrix form):

[ f1 ]   =  (E*A/L) * [  1  -1 ] [ u1 ]
[ f2 ]                 [ -1   1 ] [ u2 ]

So the element stiffness matrix is:

[k_e] = (E * A / L) * [  1  -1
                        -1   1 ]

3. Assembly — Two Elements in Series (Nodes 1–2–3)

For two bar elements connected in series: element 1 connects nodes 1–2 with stiffness k1 = E1*A1/L1, element 2 connects nodes 2–3 with stiffness k2 = E2*A2/L2. Assemble each element matrix into the global matrix by placing element entries into the rows and columns corresponding to nodal degrees of freedom. 

Global stiffness matrix [K] (3 DOF: u1,u2,u3):

[K] = [  k1      -k1       0
       -k1   k1 + k2    -k2
         0     -k2       k2 ]

4. Worked Numerical Examples (Hutton-style)

Example 2.1 — Single bar fixed at left end, axial load at right end

Problem statement:

A steel bar of length L = 2.0 m, cross-sectional area A = 100 mm², and Young's modulus E = 200 GPa is fixed at node 1 (left end) and loaded by an axial tensile force F = 10 kN at node 2 (right end). Determine the nodal displacement u2.

Step 1 — Units and constants:

A = 100 mm^2 = 100 × 10^-6 m^2 = 1.0e-4 m^2
E = 200 GPa = 200 × 10^9 Pa = 2.0e11 Pa
L = 2.0 m
F = 10 kN = 10,000 N

Step 2 — Element stiffness:

k = E * A / L
  = (2.0e11) * (1.0e-4) / 2.0
  = (2.0e7) / 2.0 = 1.0e7 N/m

Step 3 — Equilibrium equation (single DOF since u1=0):

F = k * (u2 - u1),   with u1 = 0
=> u2 = F / k = 10000 / 1.0e7 = 1.0e-3 m = 1.0 mm
Result: Displacement at free end u2 = 1.0 mm.

Example 2.2 — Two elements in series, load at middle node

Problem statement:

Two identical bar elements (element 1 and element 2) each of length L = 1.0 m, area A = 50 mm², and E = 210 GPa are connected in series (nodes 1–2–3). Node 1 is fixed (u1 = 0). A concentrated axial load F = 5 kN is applied at node 2 (the middle node). Find nodal displacements u2 and u3.

Step 1 — Convert and compute element stiffness (identical elements):

A = 50 mm^2 = 50 × 10^-6 m^2 = 5.0e-5 m^2
E = 210 GPa = 2.10e11 Pa
L = 1.0 m

k = E * A / L = (2.10e11)*(5.0e-5)/1.0 = 1.05e7 N/m

Step 2 — Global stiffness matrix [K] (3 DOFs u1,u2,u3):

[K] = [  k   -k    0
       -k  2k   -k
        0  -k    k ]
    where k = 1.05e7

Step 3 — Apply boundary conditions and loads:

Known: u1 = 0.
Loads: nodal force vector {F} = [ 0, 5000, 0 ]^T  (force applied at node 2)

Step 4 — Reduce system to unknown DOFs (u2, u3):

Reduced K (for u2,u3):
[ K_red ] = [ 2k  -k
              -k   k ]

Numerical values:
2k = 2 * 1.05e7 = 2.10e7
-k = -1.05e7
k = 1.05e7

So:
[ K_red ] = [ 2.10e7  -1.05e7
              -1.05e7  1.05e7 ]

Step 5 — Solve K_red * {u2,u3} = {5000, 0}

Write equations:
2.10e7*u2 - 1.05e7*u3 = 5000        (1)
-1.05e7*u2 + 1.05e7*u3 = 0         (2)

From (2): -1.05e7*u2 + 1.05e7*u3 = 0  =>  u3 = u2

Substitute into (1): 2.10e7*u2 - 1.05e7*u2 = 5000
=> (1.05e7)*u2 = 5000
=> u2 = 5000 / 1.05e7 = 4.7619e-4 m = 0.47619 mm

Then u3 = u2 = 0.47619 mm
Results: u2 = u3 = 0.4762 mm (approx).

Example 2.3 — Two-element bar with different cross-sections (composite bar)

Problem statement:

A composite bar consists of element 1 (left) and element 2 (right). Element 1: L1 = 1.0 m, A1 = 80 mm², E = 200 GPa. Element 2: L2 = 2.0 m, A2 = 120 mm², E = 200 GPa. Left end (node 1) is fixed. A tensile force F = 15 kN is applied at the right end (node 3). Find displacements u2 and u3.

Step 1 — Convert units and compute k1 and k2:

A1 = 80 mm^2 = 8.0e-5 m^2
A2 = 120 mm^2 = 1.2e-4 m^2
E  = 200 GPa = 2.0e11 Pa
L1 = 1.0 m
L2 = 2.0 m

k1 = E*A1 / L1 = (2.0e11 * 8.0e-5) / 1.0 = 1.6e7 N/m
k2 = E*A2 / L2 = (2.0e11 * 1.2e-4) / 2.0 = (2.4e7) / 2 = 1.2e7 N/m

Step 2 — Global stiffness matrix [K]:

[K] = [  k1     -k1     0
       -k1   k1 + k2   -k2
         0     -k2     k2 ]
Substitute k1=1.6e7, k2=1.2e7:
[K] = [ 1.6e7   -1.6e7      0
       -1.6e7  2.8e7   -1.2e7
          0    -1.2e7   1.2e7 ]

Step 3 — Apply boundary conditions and loads:

u1 = 0 (fixed). External nodal load at node 3: F3 = 15000 N
Nodal force vector {F} = [ 0, 0, 15000 ]^T

Step 4 — Reduce and solve for u2 and u3:

Reduced system (u2,u3):

[ 2.8e7   -1.2e7 ] [ u2 ]   = [ 0 ]
[ -1.2e7   1.2e7 ] [ u3 ]     [ 15000 ]

Write equations:
2.8e7*u2 - 1.2e7*u3 = 0          (1)
-1.2e7*u2 + 1.2e7*u3 = 15000     (2)

From (2): -1.2e7*u2 + 1.2e7*u3 = 15000
=> divide both sides by 1.2e7:
 -u2 + u3 = 15000 / 1.2e7 = 1.25e-3
 => u3 = u2 + 1.25e-3

Substitute into (1):
2.8e7*u2 - 1.2e7*(u2 + 1.25e-3) = 0
=> 2.8e7*u2 - 1.2e7*u2 - 1.2e7*1.25e-3 = 0
=> (1.6e7)*u2 - (1.5e4) = 0   (since 1.2e7 * 1.25e-3 = 1.5e4)
=> u2 = 1.5e4 / 1.6e7 = 9.375e-4 m = 0.9375 mm

Then u3 = u2 + 1.25e-3 = 9.375e-4 + 1.25e-3 = 2.1875e-3 m = 2.1875 mm
Results: u2 = 0.9375 mm, u3 = 2.1875 mm (approx).

5. Remarks and Teaching Notes

  • Always check units carefully before calculations (convert mm² to m², GPa to Pa, etc.).
  • When assembling global matrices, ensure correct placement of element terms according to connectivity.
  • For symmetric problems with identical elements the matrix structure often simplifies solution calculations.
  • Show the algebraic steps in class; students should be able to reproduce matrix assembly and solve reduced systems.
